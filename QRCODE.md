# 三合一收款码制作教程

## 什么是三合一收款码？

一个二维码，扫出来后可以选微信、支付宝、云闪付任意一个支付。

---

## 方法一：使用在线工具（免费）

### 步骤

1. **打开工具网站**
   - 彩虹收款码：https://codepay.hk/
   - 或者搜索"收款码合并"

2. **上传你的收款码**
   - 微信收款码
   - 支付宝收款码

3. **生成三合一码**
   - 点击生成
   - 下载图片

## 方法三：自己制作（需要一定技术）

### 使用 qrcode 库

```javascript

npm install qrcode

// 生成收款码
const QRCode = require('qrcode')

// 微信收款码内容（示例）
const wechatUrl = 'wxp://...'

// 生成图片
QRCode.toFile('payment-qr.png', wechatUrl, {

```

## 注意事项

### ⚠️ 安全提示
- 不要在网上随便上传你的收款码
- 使用可信的工具网站
- 收款码只用于收款，不要泄露太多信息

### 📱 推荐做法

1. **个人测试阶段**：用个人收款码就行
2. **正式运营后**：申请商家收款码
3. **大额收费**：考虑对接支付API
