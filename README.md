# 小小树 K12 学习平台

> 🌱 像小树一样茁壮成长

一个免费的 K12 在线学习平台，涵盖小学到高中的知识点。

## 功能特点

- ✅ 知识点浏览（人教版教材同步）
- ✅ 初中数学（七/八/九年级）
- ✅ 响应式设计（支持手机、平板、电脑）
- 🔄 持续更新中...

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- **框架**: Next.js 14
- **样式**: Tailwind CSS
- **托管**: Vercel (免费)

## 域名部署

1. 购买域名（如 xiaoxiaoshu.com）
2. 在 Vercel 绑定自定义域名
3. 配置 DNS 解析
4. 完成部署！
